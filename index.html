Dansk Låsesystem

Project involving creating a web application for "Dansk Låsesystem" which is a fictional company that creates locks that can be placed all around your house. The web application allows the user to manage locks that have been set up by a company locksmith. The user can create groups of locks, and lend them to other users such as their family members.
Test the application by creating a user and then setting up a lock for that user by logging in as user: admin@locks.dk/admin

Made using Waku (lightweight React framework) for the frontend and PHP/MySQL for the backend.

Plantmate

A project that integrates a chatbot into a recipe website using Mistral's API. The chatbot allows users to interactively search for recipes, ask cooking-related questions, and get personalized suggestions. Users can easily add discovered recipes to their personal recipe list and organize them into custom ccollections. This creates a more engaging and user-friendly experience for those looking to explore new recipes and cooking tips through natural conversations.

Created using React Router 7, MongoDB, and Mistral API.
